What is a console screen and how to use it

A console screen is a combination of a display monitor and an input device, usually a keyboard and mouse pair, which allows a user to input commands and receive visual output from a computer or computer system. ... A console is also known as a computer console, a system console or simply as a terminal. In order to open console screen you need to double click your webpage and go to the console tab at the top.
